MAXFİT GYM WEB PANEL - KURULUM

1) Bilgisayarda bu klasörü açın.
2) Terminal açın.
3) Şunları yazın:
   npm install
   npm run dev

Canlı yayın için:
1) Vercel hesabı açın.
2) Bu proje klasörünü yükleyin.
3) Deploy deyin.

Firebase bağlamak için:
1) Firebase'de proje açın.
2) Firestore Database ekleyin.
3) Proje ayarlarından web uygulaması oluşturun.
4) Verilen anahtarları src/App.jsx içindeki firebaseConfig alanına yapıştırın.
5) Tekrar build/deploy yapın.
